Conceitos CSS

CSS: sigla que significa "Cascading Style Sheets"
Serve para conferir estilo às páginas html.Com ele é possível formatar as páginas html de forma mais simples, evitando repetições.
Por exemplo, pode-se definir estilos por classes de elementos.

Estrutura CSS

SELETOR {DECLARAÇÃO: PROPRIEDADE}

EXEMPLOS:

  • Definição de cor para os elementos em h1

    h1 {background-color: black;color:white}

    Nesse caso: h1 é um seletor, background-color e color são declarações, black e white são propriedades

  • Definição por classes

    É possível também formatar um seletor específico, utilizando-se seletores de classe. Digamos que tenhamos diversos seletores h1 em um arquivo html. Podemos dividi-los em diversas classes:

    < h1 class=x > CONTEÚDO </h1 >

    < h1 class=y > CONTEÚDO </h1 >

    < h1 class=z > CONTEÚDO </h1 >

    Se quisermos mudar a aparência de todos os seletores h1 de uma vez só, escrevemos da seguinte forma no arquivo .css:

    h1{back-ground-color:black;color:white}

    Se quisermos somente alterar a classe x, devemos escrever da seguinte forma:

    .x{back-ground-color: black;color:white}

Para estabeler um link entre a página html e o arquivo css, devemos escrever o seguinte código no arquivo html:

< link rel="stylesheet" type="text/css"
href="NOME DO ARQUIVO.css" >
