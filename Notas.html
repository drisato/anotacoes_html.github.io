HTML: sigla que significa "Hypertext Markup Language" e é a linguagem utilizada para criar páginas na web.

Para a criação de documentos html é necessário utilizar um editor de texto, utilizando-se liguagem de programação. Um exemplo de editor de texto é o software "Atom".

The World Wide Web: é um conjunto de documentos html.
Cada página html se comunica com outra atavés de links ou hyperlinks.

Para acessar a internet, é necessário utilizar um navegador (ex. Google Chrome, Edge, Firefox), que se comunica com os servidores através de um protocolo de comunicação chamado HTTP (HyperText Transfer Protocol).

Como construir um arquivo html:

É necessário utilizar Elementos de HTML, que consistem em uma tag de abertura e uma de fechamento.

A estrutura é seguinte:

< Tag de abertura > CONTEÚDO < /Tag de fechamento >

EXEMPLOS:

  • <b>CONTEÚDO</b>: Deixa o texto em negrito
  • <strong>CONTEÚDO</strong>: Também deixa o texto em negrito ou destacado
  • <em>CONTEÚDO</em>: Deixa o texto em itálico
  • <h1>CONTEÚDO</h1>: Deixa o texto em destaque com letras grandes, normalmente usado para títulos. Pode ser do tipo h1, h2, h3, h4
  • <div>CONTEÚDO</div>: Cria uma caixa de texto
  • <span>CONTEÚDO</span>: Cria um elemento alinhado
  • <br>: Pula uma linha
  • <p>: Cria um parágrafo
  • <a href="www.xxx.com">Nome do site</a>: Cria um link
  • <img>: Inclui uma imagem

As Tags que não possuem tag de encerramento são chamadas de VOID Tags
